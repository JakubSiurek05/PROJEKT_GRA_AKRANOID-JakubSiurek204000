# PROJEKT_GRA_AKRANOID-JakubSiurek204000

====== Projekt w języku C++ =======
* Zamysł: Prosta gra w stylu Arkanoid, z funkcjonalną paletką, piłką z mechaniką odbijania oraz rzędami bloków o różnej ilości żyć do zbicia.
* Koniec gry: piłka spadła pod paletkę lub wszystkie bloki zostały zniszczone

* Klasy:
  - Piłka
  - Paletka
  - Stone
  - Game
  - GameSave
  - Menu

 *  Sposób działania *
    Po uruchomieniu gry pojawia się menu wyboru, w którym użytkownik może wybrać:
    - Nowa gra (Rozpoczyna rozgrywkę)
    - Wczytaj grę (Wczytuje wcześniej zapisany stan gry, aktywuje sie po kliknieciu przycisku F9)
    - Wyjście z gry (Kończy program)
